# User Management Dashboard

## Overview

A React-based User Management Dashboard implementing CRUD operations using the JSONPlaceholder API.

## Features

- View Users
- Add User
- Edit User
- Delete User
- Search Users
- Filter Users
- Sort Users
- Pagination
- Loading Indicator
- Error Handling
- Responsive Design

## Tech Stack

- React
- Vite
- Axios
- CSS

## Installation

```bash
npm install
npm run dev
```

## API

https://jsonplaceholder.typicode.com/users
